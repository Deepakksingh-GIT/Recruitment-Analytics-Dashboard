# 📊 Recruitment Analytics Dashboard

Recruitment Analytics Dashboard built using MySQL, Python, and Streamlit to analyze hiring trends, candidate performance, and salary insights with interactive visualizations.

---

## 📌 Project Overview

This project focuses on analyzing recruitment data to uncover key hiring trends and improve decision-making. The data was structured using MySQL, cleaned and transformed using Python (Pandas), and visualized through an interactive Streamlit dashboard.

---

## ❓ Problem Statement

Recruitment teams often struggle to track hiring efficiency, candidate quality, and source effectiveness. This project aims to solve these challenges by providing data-driven insights into the hiring process.

---

## 🔄 Data Pipeline

* Data stored and structured in MySQL
* Exported dataset into CSV format
* Data cleaning and preprocessing using Python (Pandas)
* Exploratory Data Analysis (EDA) performed
* Interactive dashboard built using Streamlit

---

## 🛠 Tech Stack

* Python (Pandas, NumPy)
* Data Visualization (Seaborn, Matplotlib)
* Streamlit (Dashboard)
* MySQL (Database)

---

## 📊 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Interactive Dashboard with Filters
* Hiring Trend Analysis
* Salary Insights
* Candidate Performance Analysis
* Downloadable filtered data

## 📈 Business Insights

* LinkedIn generates the highest number of successful hires
* Candidates with higher technical scores have better selection rates
* Salary increases with candidate experience
* Data Science roles require higher experience and offer higher salaries
* Efficient hiring processes reduce overall recruitment time

## 🚀 How to Run

### 1. Clone the repository

git clone https://github.com/Deepakksingh-GIT/Recruitment-Analytics-Dashboard.git
cd Recruitment-Analytics-Dashboard

### 2. Install dependencies

pip install -r requirements.txt

### 3. Run the dashboard

streamlit run dashboard/app.py

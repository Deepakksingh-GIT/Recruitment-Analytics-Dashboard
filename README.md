# Recruitment-Analytics-Dashboard
Recruitment Analytics Dashboard built using MySQL, Python, and Streamlit to analyze hiring trends, candidate performance, and salary insights with interactive visualizations.


# 📊 Recruitment Analytics Dashboard

## 📌 Project Overview
This project analyzes recruitment data to identify hiring trends, candidate performance, and source effectiveness. It provides insights to improve hiring decisions.

## 🛠 Tech Stack
- Python (Pandas, NumPy)
- Data Visualization (Seaborn, Matplotlib)
- Streamlit (Dashboard)
- MySQL (Data Storage)

## 📊 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Dashboard with Filters
- Hiring Trend Analysis
- Salary Insights
- Candidate Performance Analysis

## 📈 Key Insights
- LinkedIn is the most effective hiring source
- Higher technical scores significantly increase selection chances
- Salary increases with experience
- Data Science roles have higher hiring demand
- Most hiring is completed within a short time frame
